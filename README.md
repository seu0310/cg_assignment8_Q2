## Compilation & Run Instructions

1. Open the `assignment8_q2.sln` file using **Visual Studio 2022**.
2. Build the project:
   - From the menu: `Build > Build Solution`, or
   - Shortcut: `Ctrl + Shift + B`
3. Run the program:
   - From the menu: `Debug > Start Without Debugging`, or
   - Shortcut: `Ctrl + F5`
